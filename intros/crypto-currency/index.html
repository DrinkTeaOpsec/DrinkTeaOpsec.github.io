加密货币：支付安全的终点

前言

长久以来，支付安全一直是个棘手的问题。在网上，我们要么使用微信支付宝，要么使用银行卡进行支付。而这就给警方抓捕带来了极大的便利性：你做的所有交易他们都可以看得见，只要他们顺藤摸瓜，一点点地通过交易历史追踪下去，就可以找到你本人。

而加密货币的出现改变了这一局面：所有的帐号都是匿名的，创建帐号的时候完全不需要提供任何个人信息；进行交易时，警方只能看到一个个匿名帐号在网上做交易，无法直接辨识交易双方的身份。我们只需要简单地使用加密货币替代支付宝，就可以极大程度地避免身份暴露。

而某些加密货币则更为先进：交易时的金额和来源地址都是加密的，警方甚至连哪个帐号在进行交易都看不见，极大方便了我们的匿名活动。

注：本人非币圈人士，本人推广加密货币的原因也仅仅是促进操作安全

门罗币：支付安全的最佳选择

门罗币是一种专注于匿名性和安全性的加密货币。

特点

  • 除了转帐双方之外，没有人可以看到一笔交易的具体金额和转帐双方的账户地址

  • 即使是收款方也无法看到付款方的地址

  • 收款方可以创建8开头的临时地址（子地址），这样即使是付款方也无法得知收款方的真实地址（以4开头）

  • 转帐速度较慢（20分钟）

也就是说只要你使用临时地址，你就可以安全地通过支付宝购买门罗币而不用担心任何信息泄漏。警方不会因为你买了门罗币而得知你的门罗币地址，更不会看到你的所有交易。

使用

  • MyMonero手机客户端

    • 注意：不支持临时地址，你可能需要创建多个钱包来防止自己的信息泄漏

    • 安装：在应用商店/Github搜索即可

    • 新建钱包：

      • 打开软件，点击Create new wallet

      • 为钱包设置一个名字和颜色

      • 阅读注意事项并勾选已阅读

      • 把屏幕上出现的几十个汉字抄写下来（或者保存到密码管理器内）

        • 警告：这些汉字是你的恢复密码，如果别人拿到了这些汉字就能直接控制你的帐号

      • 在下一个页面按照顺序选择恢复密码的前七个汉字

      • 最后设置一个本地加密的密码

    • 发送门罗币：

      • 在APP中找到Send Monero界面，选择钱包，填入金额和目标地址，然后点击发送即可

      • 每次转帐需要约20分钟

    • 接收门罗币：

      • 在APP中找到收款页面，把地址或二维码发送给发送方即可

  • 门罗币电脑GUI客户端：

    • 支持临时地址

    • 新建钱包：

      • 首先选择语言和简单模式（Simple Mode）

      • 剩下的步骤同手机客户端

    • 发送门罗币：同手机客户端

    • 接收门罗币：

      • 在接收页面中点击Create new address生成一个临时地址

      • 在下方选择临时地址，之后将地址/二维码发送给付款方即可

注意

  • 如果你可以使用临时地址：建议你对每一笔交易都生成一个临时地址

  • 如果你只能使用手机客户端：建议对每一个身份生成一个钱包，在这些钱包之间转帐是安全的。

    • 为接收买来的门罗币创建一个钱包

    • 为接收朋友的转帐创建一个钱包

    • 等等

USDT: 虚拟货币交易的一般选择

USDT是一种锚定美元的虚拟货币，业内一般简称为U。因为锚定机制的存在，一个UDST和一美元等价，极大地方便了交易。

USDT依赖其他虚拟货币的网络进行交易。也就是说，你在发起USDT转帐的时候需要支付一点其他的虚拟货币作为给矿工的小费。常用的USDT转帐网络有ETH的ERC20和Tron的TRC20两种。而为了更快的转帐速度和更低的小费，我们一般在TRC20网络中使用USDT。

使用

  • imToken

    • imToken是一个手机上的虚拟货币钱包APP，可以同时管理多种虚拟货币。

    • 注册完成后，选择TRX钱包，你应该就可以看见你的USDT资产了，点击即可进行收款和转帐。

    • 每次转帐时，钱包内会扣除大约1美元的TRX, 请确保每次转帐前都有足够的TRX用于扣除。

  • 各大交易所

    • 常见的有币安，火币，欧易等等，选择一个即可。注意很多交易所（如火币）都需要实名，请注意身份泄漏的风险。

    • 我们可以在交易所购买或者出售虚拟货币，可以将一种虚拟货币兑换成另一种，也可以转入转出我们的虚拟货币。

    • 利用交易所接收USDT, 就是把USDT的转入地址当作自己的收款地址使用。而给他人转帐就是从交易所中转出USDT.

    • 在交易所中转出USDT不需要额外支付TRX, 但是需要支付大约1美元的USDT作为小费。

安全地购入虚拟货币

  • 购买门罗币：

    • 直接在localmonero.co上购买即可，支持支付宝和微信，请小心诈骗风险。

    • 使用支付宝等实名支付方式可能会导致你的门罗币帐号“被实名”，请使用子地址或者一个独立的钱包接收这类方式购买的门罗币。

  • 购买USDT和其他虚拟货币：

    • 直接在交易所购买即可，注意匿名性。

    • 当然也可以先购买门罗币再转成USDT, 这样可以避免交易所的帐号“被实名”
