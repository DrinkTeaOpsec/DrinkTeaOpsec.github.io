密码管理器

介绍

简而言之，密码管理器就是用来安全地保存密码的软件。

你只需要记住一个密码，就可以安全地保存所有平台的密码。

有了密码管理器，你就可以毫不费劲地为每个网站都随机一个超长的密码，也可以在需要输入密码时一键输入。

当然，密码管理器除了可以管理密码之外，还可以安全地保存笔记等文字信息。有些密码管理器甚至还支持TOTP和自动生成密码等高级功能。

对比

那我们为什么要使用密码管理器呢？我们来和几个常见的密码管理手段对比一下：

  • 使用人脑记忆几个常用的密码，然后重复使用。

    • 这应该是大多数人都在使用的方法了。

    • 这种方法的缺点是显而易见的，这样只能记住几个很简单的密码，记忆复杂的密码不仅容易忘，而且也不是很方便。

  • 想出一个生成密码的规律，然后使用这个规律设置平台的密码。

    • 这应该是某些自认为OpSec很好的人会使用的方法。

      • 比如说B站的这位UP主

    • 首先，这种方法的安全性完全依赖于生成密码的规律，而这个生成密码的规律不能被告诉其他人，更不能让其他人分析安全性。所以这个规律的安全性完全取决于你。

    • 而大多数人在想这个规律的时候会偷懒，他们想出的规律可能就是一个很复杂的密码加上代表平台的拼音首字母。

    • 这样的话，只要某个平台的明文密码泄漏了，他们想出的规律就会很容易地被猜出来。

  • 把某些密码保存在浏览器中

    • 浏览器对密码的加密并不可靠，其中的密码可以被很容易地提取出来。

    • 你甚至能在Github上找到提取软件

  • 把密码写在纸上

    • 不想说了，真的不怕被人看到吗。。。

  • 把密码写在手机的笔记里

    • 我真的见过有人这么做，Ta还很自豪地在网上说出来。。。

推荐

简单推荐几个密码管理器吧（这里没有广告）

即使你使用的密码管理器很烂，你的安全性也会比普通人要好得多。

  • Bitwarden

    • 普通人的最佳选择

    • 开源免费，云同步全球可用

    • 可用邮箱注册，可以私有化搭建服务端。

    • 免费版支持所有常用功能，高级版支持TOTP，私有服务端功能等同于高级版

  • KeePass

    • 最安全的密码管理器

    • 开源免费

    • 密码保存在本地（自己想办法同步）

      • 这也是它特别安全的一大原因

    • 特色功能十分丰富

  • LastPass

    • 另一个免费的密码管理器，不是很熟悉

    • 免费，但不开源

    • 使用Tor访问需要付费

    • 经常出现隐私泄漏问题（不是密码泄漏）

      • 他们的服务器源码已经泄漏过了

  • 密码键盘

    • 和国内软件契合度最好的密码管理器

    • 不开源，不免费，电脑端需要用手机扫码登陆

    • 但是它能无缝兼容任何输入框，即使是微信自带浏览器内部的文本框也可以

注意

  • 使用密码管理器时，有时候你需要从密码管理器复制密码到其他APP中。注意不要让某些会扫描剪贴板的软件读取到你的密码，而且粘贴完后记得清空剪贴板。
